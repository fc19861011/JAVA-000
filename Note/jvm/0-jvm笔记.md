# JVM学习笔记

## 一、 JVM是什么，能解决什么问题?

​		JVM是Java Virtual Machine（Java虚拟机）的缩写，JVM是一种用于计算机的**规范**，它是一个虚构出来的计算机，是通过在实际计算机上仿真模拟各种计算机功能来实现的。

​		JVM帮助编程语言屏蔽与具体平台（OS）相关的信息，使得由各中编程语言编写的编译程序只需在JVM上生成运行的字节码，就可以在多种平台上运行（WORA, Write Once Run Anywhere）。
